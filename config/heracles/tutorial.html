Create: Down To Earth

These quests serve as a way to guide and teach the Create Mod.

These quests are meant to be used along with Create's Ponder system! Confused about a component? Ponder it!
You can find Ponders in the Ponder Index, which you can find from the goggles icon in the main or pause menu. You can also find them in the tooltip for every component.

Happy Questing!
